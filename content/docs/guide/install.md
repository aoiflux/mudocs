Use the latest stable Mutant release binaries when possible.

## Packaged Binary

Download the latest stable build from the official releases page:

- https://github.com/aoiflux/mutant/releases

## Install from Source

Prerequisite: install Go.

```bash
git clone https://github.com/aoiflux/mutant
cd mutant
go install
```

The Mutant parser recognizes a small core set of keywords that define the
language's control and binding model.

## Core Keywords

- `fn`
- `let`
- `true`
- `false`
- `if`
- `else`
- `return`

## Notes

Keyword coverage should be reviewed alongside parser or grammar changes in the
upstream repository.
